Title: New Client Onboarding for Surety Bond Application

User Story

As a new surety bond company client (applicant), I want to complete an online onboarding form with my personal, ownership, company, and bond requirement details (risk to insure), so that I can submit my application quickly and begin the underwriting process.

Acceptance Criteria

Personal Information

Client can enter:

  • Full legal name
  • Email address
  • Phone number
  • Home address
  • Date of birth

Required fields must be validated before continuing.

Ownership Information

Applicant can add one or more owners/shareholders.

For each owner:

  • Full name
  • Ownership percentage
  • Role / Title
  • Address

Total ownership must equal 100%.

Company Information

Applicant can enter:

  • Legal company name
  • Operating name (optional)
  • Business address
  • Industry type
  • Years in business
  • Annual revenue
  • Number of employees

Risk / Bond Coverage Information

Applicant can enter:

  • Bond type (Bid / Performance / Labour & Material / Maintenance)
  • Obligee / Project owner
  • Contract value
  • Bond amount requested
  • Project location
  • Start date / Completion date

Submission / Workflow

  • Applicant can save progress and return later.
System validates mandatory fields before submission. Upon submission: Application status = Submitted Confirmation email sent to applicant Underwriting team receives notification Notes / Delivery Considerations Responsive design for desktop/mobile Data encrypted in transit and at rest Audit trail for submitted changes Integration point for CRM / underwriting queue Include Google Analytics tags to identify client location, bond type and contract value.
